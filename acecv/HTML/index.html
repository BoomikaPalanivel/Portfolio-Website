About Me

I began my engineering journey with a deep curiosity for technology, graduating with a B.E. in Electronics and Communication Engineering. Early on, I worked on hands-on projects, like building a solar tracking system and designing a pollution monitor for construction sites, which sparked my passion for impactful tech solutions. Interning at San Technovation and Prime Solutions helped me apply and expand my skills in languages like C, C++, Python, JavaScript, and Java. When I built a headphone website using HTML, CSS, and JavaScript, I discovered a strong interest in web development.

Since then, I’ve developed tools using React, Node.js, and Java Spring Boot. Now, as a full-stack development intern, I’m creating a portfolio and a blogging platform to showcase my work and skills, constantly driven by a commitment to functionality and user experience.
